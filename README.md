# 2D-Endless-Runner
2D Endless Runner Game

Pixel Cat Runner is a 2D endless runner game built using MonoGame (Microsoft.Xna.Framework).
You play as a cute pixelated cat dodging rocks, birds, and cacti while trying to get the highest score possible.
